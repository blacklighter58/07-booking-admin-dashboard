# Booking Desk

Демонстрационная панель управления заявками beauty-студии.

## Реализованные функции

- добавление, просмотр и редактирование заявок;
- удаление заявок и изменение статусов;
- поиск, фильтрация, сортировка и статистика;
- сохранение в `localStorage` и восстановление демо-данных;
- адаптивные мобильные карточки заявок;
- защита от небезопасного HTML-вывода.

## Технологии

HTML, CSS, vanilla JavaScript, `localStorage`.

## Локальный запуск

```bash
npx serve .
```

После запуска откройте адрес, который покажет команда (обычно `http://localhost:3000`).

## Опубликованная версия

[Booking Desk](https://07-booking-admin-dashboard.vercel.app/)

## Честные ограничения

- нет backend и серверной базы;
- нет авторизации;
- данные сохраняются только в текущем браузере;
- это демонстрационный проект.
