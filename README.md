# Booking Admin Dashboard

Демонстрационная адаптивная админ-панель для небольшой beauty-студии. Помогает администратору просматривать заявки с сайта, искать записи и отслеживать их статусы.

## Текущие функции

- статистика по заявкам, рассчитанная из данных;
- таблица заявок с клиентами, контактами, услугами, датами и статусами;
- поиск по имени, телефону и услуге;
- фильтрация по статусу и сброс фильтров;
- отображение количества результатов и пустого состояния;
- действие для просмотра номера заявки;
- адаптивная навигация и мобильное меню.

## Технологии

- HTML5;
- CSS3;
- JavaScript без фреймворков и внешних зависимостей.

## Локальный запуск

Скачайте или клонируйте проект, затем откройте файл `index.html` в браузере. Установка зависимостей и сборка не требуются.
